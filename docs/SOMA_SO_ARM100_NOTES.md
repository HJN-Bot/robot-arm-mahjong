# SOMA SO-ARM100（初步产品信息 & 接入建议）

> 来源：你在群里发的产品规格截图（SO-ARM100 / 与 LeRobot 适配）。

## 1) 关键规格（对我们影响最大的）

- 自由度：5
- 夹爪：柔性自适应平行夹爪（TPU材质）
- 关节电机：STS3215 总线伺服舵机
- 传感器：关节角度绝对编码器
- 通信协议：UART
- 控制方式：外接终端（电脑、开发板等）
- 电源：3–12V / 10A 可调压电源
- 腕部相机：彩色 200 万像素、120° 广角、自动对焦、USB 免驱相机
- 底座相机：彩色 1200 万像素、125° 广角、自动对焦、USB 免驱相机

## 2) 对我们“机械臂麻将”的直接结论

- **控制侧**：很可能是“PC/开发板 → 串口（UART） → 总线舵机控制板/舵机链路”。
- **视觉侧**：自带 USB 相机意味着我们可以快速做桌面定位/目标点定位。
- **供电侧**：要认真对待电源稳定性与线材，否则动作会抖、掉电、复位。

## 3) 接入建议（最小风险路径）

### 方案 A（首选）：LeRobot 作为 Arm Adapter
- 用 `lerobot` Python 包控制 SO-ARM100/101 系列
- 我们自己只定义统一动作原语：home/grasp/look/place/tap/nod/shake/estop
- OpenClaw 通过 HTTP/CLI 调用 Python

优点：复用社区生态与驱动，Hackathon 更稳。

### 方案 B：直接串口控制（UART）
- 需要拿到：波特率、关节 ID、指令集、校准流程
- 工程量更大，不建议作为 Hackathon 主路径

## 4) 下一步需要 SOMA 官方确认/拿到的东西

- SDK/示例代码（最好 Python）
- 串口参数（baudrate 等）
- 关节 ID / 机械臂模型参数（DH 或等效）
- 校准流程（零位、软限位、夹爪开合范围）
- 急停/错误码/复位策略
