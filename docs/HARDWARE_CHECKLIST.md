# Hardware Checklist｜机械臂麻将（已有一台 SOMA SO-ARM100）

> 目标：两天内不被“线材/供电/支架/相机”这种小事卡死。

## 0) 已有硬件
- ✅ SOMA SO-ARM100 机械臂（含夹爪、线缆、供电）

---

## 1) 必备（强烈建议开工前全部到位）

### 1.1 计算与连接
- 机械臂主机：**Mac**（运行 Python + LeRobot + 本地 TTS + Orchestrator）
- OpenClaw 主机：**EC2**（Brain/编排/Discord）
- 同一 WiFi 环境（现场路由器/手机热点都可以）

### 1.2 相机与机位
- 优先使用机械臂自带相机（USB 免驱）：
  - 腕部相机（2MP）
  - 底座相机（12MP）
- 如果自带相机不方便：准备一只额外 USB 摄像头作为备份

### 1.3 桌面与道具（保证可复现）
- 一张稳定桌子（不要晃）
- 桌面标定物：
  - 推荐：**AprilTag 打印纸**（4 个角 + 中央区域标识）
  - 备选：四角高对比色贴纸
- 中央弃牌区：用胶带在桌面贴出矩形区域（视觉/放置都需要）
- 麻将牌（建议准备 20~40 张即可）

### 1.4 供电与线材（最容易翻车）
- 可调压电源（如果套装不含或不稳定，建议备一个）：3–12V / ≥10A
- Type-C 数据线若干（至少 2 根备用）
- USB Hub（可选但建议）：相机 + 机械臂 + 外设同时插 Mac
- 延长线/插排（现场经常缺）

### 1.5 安全
- 紧急停止方案（软急停 + 物理断电预案）
- 演示区域边界贴纸（避免手伸进工作区）

---

## 2) 可选增强（提升观感/产品化）
- 3D 打印外壳（不影响关节运动范围）
- 灯光（让视频更好看）
- 小音箱（播 TTS 更有存在感）

---

## 3) 开工前 10 分钟检查清单
- [ ] 机械臂能 home、能 estop
- [ ] 相机能在 Mac 上识别（Preview/ffmpeg/你的脚本能看到画面）
- [ ] 桌面标定物贴好，中央区域定义好
- [ ] 线材不绊人，供电稳定
- [ ] 现场演示脚本打印/备忘录可随时打开
