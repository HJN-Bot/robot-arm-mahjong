# Scenes Brainstorm｜机械臂麻将（OpenClaw Brain 有灵魂）

> 目的：把“会捣蛋/抽象/麻将高手”的想象，变成可执行的场景与可演示瞬间。

## A) 观众 3 秒能懂的（强烈建议优先）
1. **验牌宣告**：抓→看→扔→说“我要验牌”（Scene1）
2. **点三点**：验牌后对着人点三下（挑衅/认可）
3. **点头/摇头**：像人一样给出判断（牌没问题/你在搞）

## B) “Brain 有灵魂”的加分项（不加也能交付）
4. **台词风格切换**：`/style polite|meme` 立刻换人格
5. **安全等级**：`/safe on` → 动作变慢、话术变礼貌
6. **小小暂停**：机械臂“回收停顿 0.3s”显得在思考

## C) 进阶：麻将高手方向（需要更多规则/识别）
7. **解说型**：扔牌后说“我在做 X（理由）”（解释策略）
8. **学习型（Cron）**：每天学 1 条桌游/麻将梗/技巧，演示时随机抽一条说

## D) 现场可控的开关（建议做成指令）
- `/scene 1|2|3|4`
- `/style polite|meme`
- `/safe on|off`
- `/status`

## E) 台词库建议（两套）
- polite："我要验牌" / "牌没有问题" / "我再确认一下"
- meme："我要验牌" / "你确定吗" / "别急"（梗版谨慎使用，现场可一键禁用）
